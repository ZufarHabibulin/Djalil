# Djalil Knowledge Base

This repository contains structured data and development tracking for Djalil.

## Purpose
To maintain a single source of truth for:
- Health metrics
- Vision tracking
- Screen time and physical activity
- Education progress
- Long-term development planning

## Structure
- data/ — structured JSON data
- metrics/ — measurable indicators over time
- plans/ — strategic plans
- logs/ — observations and notes

This repository is designed to be readable by both humans and AI systems.
